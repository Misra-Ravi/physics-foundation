Reference — Conversions & Rules

Absolute → Fractional: Δa / a
Fractional → Absolute: (Δa/a) × a
Fractional → Percent: (Δa/a) × 100%
Percent → Fractional: % / 100
Rounding reminder: Match decimal places — if the uncertainty has more d.p., round the number; if fewer d.p., round the uncertainty. Unit after the uncertainty = absolute. Unit after the number only = fractional.
