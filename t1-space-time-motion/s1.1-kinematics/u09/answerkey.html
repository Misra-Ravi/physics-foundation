Part B — True or False

B1 FALSE. a = 0 means constant velocity (which could be any value, including non-zero). The object is stationary only if v = 0.
B2 TRUE. ΔV = a × t = −4 × 3 = −12 ms⁻¹. ✓
B3 FALSE. Zero slope on a V-T graph means a = 0, so the A-T graph shows a horizontal line ON the x-axis (a = 0), not at a non-zero value.
B4 TRUE. Total area = total ΔV. If ΔV = 0, then v_final = v_initial.
