Part B — True or False

FALSE
Distance is a scalar (no direction). It is not a vector.
TRUE
Displacement = x_f − x_i. If start = end, then s = 0 regardless of path.
FALSE
Displacement can never be larger than distance. Speed is irrelevant. The straight-line is always the shortest path.
TRUE
Variable: s, unit: metres (m).
