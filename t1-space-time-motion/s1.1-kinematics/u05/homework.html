Complete all parts. Show full working for Part A. Correct any false statements in Part B.

Reference — Key Formulas and Rules

Velocity
v = Δd / Δt = Rise / Run
Y-intercept
= Starting position (P₀)
Displacement
Δd = P_final − P_initial
Distance
Sum of all |segment changes|
Homework Tip

Always read the y-intercept first (starting position), then identify each segment before calculating velocity. For distance, use magnitude (no negatives). For displacement, subtract start from finish.
